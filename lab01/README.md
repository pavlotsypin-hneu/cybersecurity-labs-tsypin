Аналізатор безпеки пароля - проста консольна Python-програма, яка аналізує надійність пароля з урахуванням персональних даних користувача. 
Скрипт допомагає виявити слабкі місця пароля та пропонує конкретні рекомендації для його посилення.

МОЖЛИВОСТІ ПРОГРАМИ:
-Перевірка на збіги з персональними даними:
1. ім’я, прізвище, по-батькові;
2. фрагменти дати народження (рік, останні дві цифри, день+місяць тощо).
-Оцінювання складності пароля за критеріями:
1. довжина;
2. різноманітність символів (великі/малі літери, цифри, спеціальні символи);
3. наявність словникових слів.
-Виявлення слабких місць: словникові слова, персональні збіги, надто проста структура.
-Формування детального звіту з оцінкою у діапазоні 1–10.
-Генерація рекомендацій, як зробити пароль надійнішим.

Використані технології: 
1. Python 3.13
2. Модулі: re, datetime
3. Консольний інтерфейс введення користувача

?Як запустити?
python main.py


Далі програма попросить ввести:
1. пароль;
2. ПІБ користувача (через пробіл);
3. дату народження у форматі DD.MM.YYYY / DD-MM-YYYY / YYYY-MM-DD.


Програма визначить, що пароль містить ім’я та рік народження, і надасть рекомендації.
