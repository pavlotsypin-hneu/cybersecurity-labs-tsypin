# Спрощена система цифрових підписів

Навчальна програма для демонстрації принципів асиметричних цифрових підписів без використання складних криптографічних алгоритмів.

## Мета
Показати, як працює цифровий підпис: створення пари ключів, підписання файлу та перевірка підпису, а також виявлення змін у документі.

## Технології
- Python  
- hashlib (SHA-256)  
- Робота з файлами  

## Опис роботи
Приватний ключ генерується як хеш від персональних даних користувача. Публічний ключ математично пов’язаний з приватним. Для підпису файлу обчислюється його хеш, який поєднується з приватним ключем. Перевірка підпису виконується шляхом повторного обчислення хешу та порівняння результатів.

1. Запустити програму:
   ```bash
   python main.py
Згенерувати ключі (пункт меню 1)

Підписати файл (пункт меню 2)

Перевірити підпис або продемонструвати підробку (пункти 3–4)

Результат
Програма визначає, чи є підпис дійсним, та коректно виявляє підробку у разі зміни вмісту файлу. Реалізація має демонстраційний характер і призначена для навчальних цілей.
