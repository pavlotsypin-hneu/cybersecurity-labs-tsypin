# Email-шифратор (симетричне шифрування)

Навчальна програма для демонстрації принципів захищеної комунікації між користувачами з використанням симетричного шифрування.

## Мета
Показати процес шифрування та розшифрування електронних повідомлень, генерацію ключа з персональних даних і обмін зашифрованими повідомленнями між двома користувачами.

## Технології
- Python  
- hashlib (SHA-256)  
- base64  

## Опис роботи
Ключ шифрування генерується на основі персональних даних користувача за допомогою хеш-функції SHA-256. Повідомлення шифрується за допомогою операції XOR з псевдо-ключовим потоком, після чого результат кодується у формат Base64 для зручної передачі. Розшифрування виконується у зворотному порядку з використанням того ж ключа.

## Запуск
1. Запустити програму:
   ```bash
   python main.py

2. Згенерувати ключ (пункт меню 1)

3. Зашифрувати повідомлення (пункт меню 2)

4. Розшифрувати повідомлення або переглянути демонстрацію обміну (пункти 3–4)
