## Опис проєкту

Програма демонструє роботу двох класичних алгоритмів шифрування:
- **Шифр Цезаря** - моноалфавітний шифр підстановки з циклічним зсувом літер
- **Шифр Віженера** - поліалфавітний шифр з використанням ключового слова

## Основні можливості

- Шифрування та розшифрування текстів українською мовою
- Генерація ключів на основі персональних даних (дата народження, прізвище)
- Частотний аналіз зашифрованих текстів
- Криптоаналіз методом brute force для шифру Цезаря
- Розрахунок індексу відповідності (Index of Coincidence)
- Візуалізація результатів у вигляді графіків
- Порівняльна таблиця характеристик обох методів

## Використання

1. Відкрийте файл `cipher_comparison.py`
2. Змініть персональні дані у функції `main()`:
```python
birth_date = "ДД.ММ.РРРР"  # Ваша дата народження
surname = "Прізвище"        # Ваше прізвище
```
3. Запустіть програму:
```bash
python main.py
```

## Структура коду

Класи

**`CaesarCipher`**
- Реалізація шифру Цезаря
- Методи: `encrypt()`, `decrypt()`
- Підтримка великих та малих літер української абетки

**`VigenereCipher`**
- Реалізація шифру Віженера
- Методи: `encrypt()`, `decrypt()`
- Циклічне використання ключового слова

**`CryptoAnalyzer`**
- Інструменти для криптоаналізу
- Частотний аналіз текстів
- Brute force атака на шифр Цезаря
- Розрахунок індексу відповідності

## Функції

- `generate_keys_from_data()` - генерація ключів з персональних даних
- `compare_ciphers()` - порівняльний аналіз обох шифрів
- `visualize_results()` - візуалізація частотного розподілу

## Результати роботи
