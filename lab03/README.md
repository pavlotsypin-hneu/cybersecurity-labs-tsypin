# LSB Steganography (Python)

Навчальна реалізація приховування текстової інформації в зображенні методом LSB без використання готових бібліотек стеганографії.

## Мета
Продемонструвати принцип роботи LSB-стеганографії: приховування тексту в молодших бітах пікселів зображення та його подальше витягування.

## Технології
- Python  
- Pillow (PIL)  
- NumPy  

## Опис роботи
Текстове повідомлення перетворюється у двійковий код та послідовно записується в молодші біти каналів RGB зображення. Для коректного зчитування використовується маркер кінця повідомлення. Зворотний процес дозволяє відновити прихований текст.

## Запуск
1. Помістити файл `original.png` у директорію з програмою  
2. Запустити:
   ```bash
   python main.py
